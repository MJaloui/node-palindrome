# ↔️ Server Side Palindrome Checker

**Project Overview**
This is a full-stack web application that validates whether a given string is a palindrome using server-side processing. Built with Node.js, the application demonstrates fundamental backend development concepts by utilizing the fs and http modules to handle user requests and perform palindrome validation on the server rather than the client. This project showcases the basics of creating a simple web server and processing data server-side.

**Core Features**
* Server-side palindrome validation logic
* HTTP server built with Node.js native modules
* Form submission and data processing
* Real-time validation feedback

**Technologies**
* **HTML:** Structure and form interface
* **CSS:** Styling and visual presentation
* **JavaScript (Client-side):** Form handling and user interactions
* **Node.js:** Server-side runtime environment
* **HTTP Module:** Native Node.js server creation


**Key Takeaways**
* Building a **Node.js server** using native http module
* Working with **server-side logic** and request handling
* Understanding **client-server architecture** fundamentals
* Creating **RESTful endpoints** for data validation
* Handling **HTTP requests and responses**
